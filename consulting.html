클라우드 마이그레이션 컨설팅

On-Premise 환경에서 AWS, Azure, GCP로의 대규모 마이그레이션을 계획부터 실행까지 지원합니다. 7R 전략(Rehost, Replatform, Refactor 등)을 적용하여 최적의 마이그레이션 경로를 설계합니다.

  • 마이그레이션 평가 및 전략 수립
  • 워크로드 분석 및 우선순위 결정
  • 다운타임 최소화 전략
  • 마이그레이션 후 최적화
레퍼런스: 아모레퍼시픽 On-Premise to AWS 전환 (SA 역할)

멀티클라우드 아키텍처 설계

AWS, Azure, GCP를 조합한 최적의 멀티클라우드 아키텍처를 설계합니다. 비용 효율성, 가용성, 보안 요구사항을 종합적으로 고려한 엔터프라이즈급 설계를 제공합니다.

  • 멀티클라우드 전략 컨설팅
  • 고가용성 아키텍처 설계
  • 비용 최적화 분석
  • 보안 및 컴플라이언스 설계
레퍼런스: 한국은행 CBDC 클라우드 인프라 설계

DevSecOps 파이프라인 구축

Kubernetes, Docker, Terraform을 활용한 CI/CD 파이프라인과 컨테이너 기반 인프라를 구축합니다. 보안이 내장된 자동화 워크플로우를 설계합니다.

  • Kubernetes 클러스터 설계 및 운영
  • Terraform IaC 파이프라인 구축
  • GitLab CI/CD 자동화
  • 보안 스캔 및 모니터링 통합
레퍼런스: SKT Tdeal AWS Terraform 구축, MOIM 메타버스 DevOps

Oracle DB 최적화 & 마이그레이션

13년의 Oracle 전문 경험과 OCP 자격을 바탕으로, Oracle 데이터베이스 성능 최적화 및 클라우드 DB로의 마이그레이션을 지원합니다.

  • Oracle RAC/ASM 구축 및 최적화
  • Oracle → AWS RDS/Aurora 마이그레이션
  • ZDLRA 백업 솔루션 구축
  • 데이터베이스 튜닝 및 성능 분석
레퍼런스: 신한DS, KB카드, 국민연금, 수협 ZDLRA 구축/운영
